- Rotas de Registro e Autenticação:

✅ registerUser: Rota para registrar um novo usuário na plataforma. Inclui dados básicos como nome, e-mail, senha e perfil de acesso.

sendVerificationEmail: Envia um link de verificação para o e-mail do usuário recém-registrado, garantindo a validação da conta.

verifyAccount: Valida a conta do usuário a partir do link/token enviado por e-mail.

✅ login: Realiza a autenticação do usuário, retornando um token JWT para acesso aos recursos protegidos.

refreshToken: Gera um novo token de acesso a partir de um refresh token válido.

✅ logout: Invalida o token de autenticação do usuário, encerrando sua sessão.

- Rotas de Usuários (CRUD principal):

✅ getUsers: Retorna a lista de usuários cadastrados, com suporte a paginação, filtros e busca.

✅ getUserById: Retorna os detalhes de um usuário específico pelo ID.

✅ updateUser: Atualiza dados de um usuário, como nome, e-mail, senha ou permissões.

✅ deleteUser: Remove permanentemente um usuário do sistema.

- Rotas de Perfil e Preferências:

getProfile: Retorna os dados do perfil do usuário autenticado.

updateProfile: Atualiza informações do perfil do próprio usuário, como avatar, preferências ou dados pessoais.

changePassword: Permite ao usuário alterar sua senha de forma segura.

uploadAvatar: Faz upload e atualização da imagem de perfil do usuário.

- Rotas de Permissões e Papéis:

assignRole: Atribui um papel (role) a um usuário, como admin, manager ou user.

getRoles: Retorna a lista de papéis disponíveis no sistema.

updateRole: Atualiza as permissões associadas a um papel específico.

revokeRole: Remove um papel previamente atribuído a um usuário.

- Rotas de Segurança e Auditoria:

getLoginHistory: Retorna o histórico de logins de um usuário, incluindo IP, data e dispositivo.

lockUser: Bloqueia temporariamente um usuário, impedindo login até desbloqueio.

unlockUser: Desbloqueia o acesso de um usuário previamente bloqueado.

getAuditLogs: Retorna registros de auditoria das ações realizadas pelos usuários no sistema.

- Rotas de Relatórios e Estatísticas:

getUserStats: Retorna estatísticas de usuários, como quantidade de novos registros no mês ou usuários ativos.

getActiveSessions: Lista todas as sessões ativas de um usuário no sistema.

getInactiveUsers: Retorna usuários inativos em um determinado período.
